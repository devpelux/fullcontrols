# FullControls  
![FullControls](https://raw.githubusercontent.com/devpelux/fullcontrols/master/FullControls/Icons/FullControls.png)  
<br>
Set of controls that update standard controls to be fully customizable.  

### Controls included:  
+ FullButton  
+ FullComboBox  
+ FullComboBoxItem  
+ FullPasswordBox  
+ FullTextBox  
+ FullToggleButton
